### Project 1 — Excel — Student Performance Analysis

**Excel File:** [Download Student Dataset](Files/Excel/student_grades.xlsx)

**Problem:**
Using the student dataset I analysed performance by:
- converting the data to an Excel table using Ctrl+T
- applying filters and sorting to find the best students per subject
- calculating average scores per student using `=AVERAGE()`
- finding the highest score per student using `=MAX()`
- sorting by average and highest score
- applying conditional formatting to highlight the highest and lowest averages in green and red

**Approach Taken:**
Cleaned and structured the data, applied Excel formulas across all rows, 
used conditional formatting to make results immediately visual, and sorted 
the table to rank students by performance.

**Insights and Impact:**
The analysis clearly identified top and bottom performers across subjects, 
showed gender-based performance differences, and produced a ranked summary 
that could directly support academic decision making.

### Project 6 — Pandas — Student Data Analysis


**Dataset:** student_in_.csv (35 students, 5 columns)

**Problem:**
Using Pandas I was required to load, explore, slice, manipulate, 
aggregate, and export the student dataset. This included reading data, 
selecting columns, filtering rows, creating new columns, grouping values, 
building pivot tables, sorting results, and exporting the final DataFrame.

**Approach Taken:**
Imported Pandas, loaded the CSV into a DataFrame, explored structure and 
statistics using info() and describe(), performed indexing and slicing, 
created a grade column using a custom function with apply(), grouped data 
using groupby(), built a pivot table, sorted results, and exported to CSV.

**Insights and Impact:**
Class Six had the highest average score (82.57) while Class Four had the 
lowest (68.75). Female students averaged slightly higher than males. 
Kenn Rein achieved the top score of 96. These operations form the 
foundation of real-world Python data workflows.
