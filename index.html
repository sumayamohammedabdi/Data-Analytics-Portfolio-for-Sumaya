Project 1

Excel — Data Analysis & Reporting

Analysing retail sales and student performance datasets using Excel formulas, conditional formatting, and pivot tables.

PROJECT 01 — EXCEL

Retail Sales & Student Performance Analysis

Excel tables, formulas, conditional formatting, pivot tables

Excel Pivot Tables Conditional Formatting SWITCH

Task

Analyse retail sales and student performance datasets using Excel. Convert raw data into structured tables, apply formulas to calculate averages and highest scores, and use conditional formatting and pivot tables to surface key insights.

Approach Taken

  • Converted datasets to named Excel tables using Ctrl+T
  • Applied sorting and filtering to rank students per subject
  • Used =AVERAGE() and =MAX() across all rows
  • Applied green-red conditional formatting to highlight averages
  • Built pivot table from Bike Sales data by county and product
  • Used IF/SWITCH to categorise sales as High, Medium, Low

Key Insights

  • Best student by average: Carol (81.67) — Linda achieved the highest individual score (90)
  • Cornwall and Essex led county sales at 1100 — categorised as High
  • Class Six returned the strongest class average in the Bike Sales analysis
  • Conditional formatting made top and bottom performers immediately visible

Project 2

Tableau — Global Health Insights Dashboard

Interactive Tableau dashboard exploring life expectancy, BMI, and population trends across continents and countries.

PROJECT 02 — TABLEAU

Global Health Insights Dashboard

GapminderHealth dataset — 158 countries

Tableau Dashboard Line Chart Scatter Plot Map

Task

Build an interactive Tableau dashboard showing life expectancy by continent, life expectancy trends over time, BMI correlations, and population distribution by country and gender — all in one connected dashboard.

Approach Taken

  • Built bar chart for life expectancy by continent
  • Built line chart filtered to 5 top countries from 1985–2006
  • Created scatter plot comparing BMI vs life expectancy
  • Built population map with sized and coloured circles
  • Added pie chart for gender distribution with filters
  • Combined all sheets into one interactive dashboard

Key Insights

  • Oceania and Europe have the highest average life expectancy — Africa the lowest
  • Japan shows the steepest life expectancy growth from 1985 to 2006

Project 4

SQL — Database Querying & Analysis

Writing SQL queries to filter, aggregate, join, and categorise data from relational databases.

PROJECT 04 — SQL

Relational Database Design & SQL Querying

WHERE, JOIN, GROUP BY, ORDER BY, CASE statements

SQL JOIN GROUP BY CASE Filtering

Task

Write SQL queries to retrieve, filter, sort, and combine data from relational databases. Demonstrate understanding of primary keys, foreign keys, JOIN types, and aggregation functions.

Approach Taken

  • Used WHERE clauses for targeted filtering
  • Applied ORDER BY to sort query results
  • Used BETWEEN to retrieve values within ranges
  • Used JOIN statements to combine related tables
  • Used GROUP BY with COUNT, AVG, and SUM
  • Wrote CASE statements to create conditional categories

Key Insights

  • IN operator is cleaner than multiple OR conditions when filtering multiple values
  • CASE statements replicate Excel IF logic directly inside SQL queries
  • GROUP BY with AVG and COUNT enables instant aggregated business reporting
  • Understanding primary and foreign keys is what makes JOINs return correct results
